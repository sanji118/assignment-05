How do you select an element from the DOM?

Here are some methods , you can use the following methods depending on your requirements: 1. document.querySelector(): Selects the first element that matches a CSS selector.
2. document.querySelectorAll(): Selects all elements that match a CSS selector.
3. document.getElementsByClassName(): Selects all elements with a specific class name.
4. document.getElementById(): Selects an element by its ID.
5. document.getElementsByTagName(): Selects all elements with a specific tag name.

What is event delegation in the context of the DOM, and why is it useful?

Event delegation refers to the process of using event propagation (bubbling) to handle events at a higher level in the DOM than the element on which the event originated. It allows us to attach a single event listener for elements that exist now or in the future.When an event occurs on an element, it first triggers the event on that specific element (the target), but then the event bubbles up (propagates) through its ancestor elements in the DOM tree. By using event delegation, you take advantage of this event propagation to handle events for child elements from a common parent.

How do you manipulate an element's attributes and styles using the DOM?

Attributes are values that contain additional information about HTML elements. They usually come in name/value pairs, and may be essential depending on the element. Some of the most common HTML attributes are the src attribute of an img tag, the href of an a tag, and the class, id, and style attributes. For a full list of HTML attributes, view the attribute list on the Mozilla Developer Network. Custom elements that are not part of the HTML standard will be prepended with data- or aria-. In JavaScript, we have four methods for modifying element attributes: hasAttribute() getAttribute() setAttribute() removeAttribute()
